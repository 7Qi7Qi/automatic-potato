**数字表示单词**
----
+ 手机的九宫格输入法中，输入数字的键位是可以和字母键位对应的。按如下将单词用一串数字来进行表示。
````
    +------+-----+------+
    |   1  |  2  |   3  |
    |   -  | ABC |  DEF |
    +------+-----+------+
    |   4  |  5  |   6  |
    |  GHI | JKL |  MNO |
    +------+-----+------+
    |   7  |  8  |   9  |
    | PQRS | TUV | WXYZ |
    +------+-----+------+
           |  0  | 
           |  -  | 
           +-----+
````

+ 举例：
  + 输入：cat（不区分大小写）
  + 输出：228
